Installing Astropy

If youre using an Anaconda environment, use the follwing command after replacing it with correct environment name.

conda install -n environmnet_name -c astropy astropy

If you're using the default Anaconda environment, use

conda install -c astropy astropy

File handling using astropy

from astropy.io import fits

file_path = "sample/09feb2021_bias1.fits"
hdu_list = fits.open(file_path)
print (hdu_list)

[<astropy.io.fits.hdu.image.PrimaryHDU object at 0x7f5f4c062cf8>]

In the default mode "open" function is readonly. The open function returns an object called an HDUList.
HDUList consist of collection of HDU (Header Data Unit) objects. It is the highest componet of fits file
structure. It can be consist of a header and a data array or a table
